## 2021 날씨 빅데이터 콘테스트
- **주최**: 기상청
- **분석 목적**: 날씨에 따른 소비패턴 분석
- **해결 과제**: 날씨에 민감한 상품군 분석, 소비패턴 트렌드 분석, 날씨에 따른 수요 예측, 결과 활용 방안 등
- **웹페이지**: [날씨 빅데이터 콘테스트](https://bd.kma.go.kr/contest/main.do)
